# Student-Course_Enrollment-System

## Project Description

This project is a simple web application developed using PHP and MariaDB on Raspberry Pi Zero 2W.

The system allows administrators to:
- Add students
- Add courses
- Enroll students into courses
- Update enrollment information
- Remove enrollments

The project demonstrates:
- CRUD database operations
- Relational database design
- 3-tier web architecture
- PHP server-side scripting

## Technologies
- PHP
- MariaDB
- HTML/CSS
- Apache2

## Hardware Platform
- Raspberry Pi Zero 2W

## Planned Features
- Student management
- Course management
- Enrollment management
- Member introduction page
